
File: 
	winc1500_fw_19.5.2_rc7_setup.exe

Description: 
	Windows installer to install the files required for completing the WINC1500 firmware update through UART serial connection.	

System Requirement:
	Wincdows 7 or later with USB/Serial cable connecting to the WINC1500 development board.

Remark:
	Install the package at a high-level directory (e.g. C:\winc1500); do not install at the default location of
	C:\Program Files (x86)\...  as the directory paths will be too long.

Reference:
	Refer to "WINC1500 Firmware Update Guide" section in the Harmony Help 2.0.3 and later, for 
	additonal system and configuration requirements and how-to instruction.
